# meterm-frontend

### Steps to run the app.

#### Prerequisites
- ```Node.js```
- ```Npm or Yarn```

<br>

Clone the repo
```
git clone https://github.com/divinirakiza/meterm-backend
```
Install dependencies
```bash
npm i
```
or
```bash
yarn
```

Run the server
```
npm run dev
```
or
```
yarn dev
```

Run tests
```
npm run test
```
or
yarn test
```
